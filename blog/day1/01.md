# 上机
上机题目有点太坑了，OJ经常崩溃，还PHP5.5，ngx1.4.1想笑。
后来T2老超时，写T6，结果打表暴力时出了bug，调试输出太多就直接freopen重定向到123.txt。
**结果123.txt占了`96.5`MB硬盘!**
索性写了一个占满磁盘的程序，大概像下面：
```
#include <iostream>
#include <cstdio>

using namespace std;

int main(){
    freopen("123.txt","w",STDOUT);
    while(true){
        cout<<"[一堆乱码]";
    }
    return 0;
}
```
最后搞出一个10G的文件。。。
